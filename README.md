# A/B Testing Dashboard with Streamlit

This project analyzes user conversion data using A/B testing and visualizes insights via Streamlit.

## Features
- Data cleaning and EDA
- Conversion analysis by test group
- Interactive Streamlit UI

## How to Run
```bash
streamlit run st.py
```
